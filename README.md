## Cấu trúc tổ chức app

```
app.js
public
    css
    img
routes
    ...js
views
    ...html
```

Thư mục ```routes``` chứa api

Thư mục ```views``` chứa html

Thư mục ```publc``` chứa static file (img, css, ...)


## Hướng dẫn deploy trên Heroku

Tạo app
```
heroku create
```

Deploy lên Heroku
```
git push heroku master
```
